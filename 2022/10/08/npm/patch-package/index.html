|йҳ…иҜ»йҮҸ:

жҲ‘д»¬еңЁејҖеҸ‘дёӯ,з»Ҹеёёз”ЁеҲ°з¬¬дёүж–№дҫқиө–,еҒҮеҰӮжҲ‘д»¬зҺ°еңЁз”ЁеҲ°дәҶelement-uiзҡ„buttonз»„д»¶,дҪҶжҳҜеҸ‘зҺ°жңүж—¶еҖҷйңҖиҰҒжӢ“еұ•,жҲ–иҖ…ж”№з»„д»¶bug, иө·еҲқжҲ‘д»¬жғіеҲ°зҡ„ж–№жЎҲжҳҜ:

  • жүҫеҲ°node_modulesжәҗз Ғ,еӨҚеҲ¶еҲ°жң¬ең°дҝ®ж”№,з„¶еҗҺеј•з”Ёжң¬ең°,дҪҶжҳҜиҝҷз§ҚдёҖжҳҜдёҚеӨӘдјҳйӣ…,дәҢжҳҜеҰӮжһңдәәе®¶жӣҙж–°,иҮӘе·ұйЎ№зӣ®иҝҳиҰҒйҮҚж–°еӨҚеҲ¶иҝҮжқҘдҝ®ж”№,йқһеёёйә»зғҰ.

жңүжІЎжңүе…¶д»–ж–№жЎҲе‘ў?

д»Җд№ҲжҳҜpatch-package?

е®ҳж–№иҝҷж ·жҸҸиҝ°: patch-package lets app authors instantly make and keep fixes to npm dependencies. ItвҖҷs a vital band-aid for those of us living on the bleeding edge.

з®ҖеҚ•жқҘиҜҙе°ұжҳҜжү“иЎҘдёҒгҖҒеҲӣеҸҜиҙҙ

з”Ёжі•

  1. npm install patch-package --save-dev ,е…Ҳе®үиЈ…еҢ…

  2. зӣҙжҺҘеңЁnode_modulesдёӯдҝ®ж”№еҢ…ж–Үд»¶,з„¶еҗҺиҝҗиЎҢ npx patch-package еҢ…еҗҚ,
    жӯӨж—¶patch-packageдјҡеңЁйЎ№зӣ®ж №зӣ®еҪ•з”ҹжҲҗдёҖдёӘpatchesж–Үд»¶еӨ№,е№¶з”ҹжҲҗиЎҘдёҒж–Үд»¶,ж–Үд»¶еҗҚж јејҸдёә еҢ…еҗҚ+еҢ…зүҲжң¬еҸ·з Ғ еҰӮelement-ui+2.15.10
    (жіЁж„ҸеҰӮжһңnode_modulesдёӯеҢ…ж–Үд»¶жІЎжңүж”№еҠЁ,иҝҗиЎҢе‘Ҫд»Ө,дјҡжҸҗзӨәжІЎжңүж”№еҠЁ).

  3. еңЁpackage.jsonж–Үд»¶дёӯзҡ„ scripts дёӯеҠ е…Ҙgitй’©еӯҗ "postinstall": "patch-package"гҖӮиҝҷж ·жҲ‘д»¬жҜҸж¬Ўnpm installеҗҺ, patch-packageдјҡеҺ»жүҫpatcheж–Үд»¶е№¶е°Ҷдҫқиө–иҝӣиЎҢжӣҙж–°,еҰӮжһңеҗҢдјҙдёҚжғіеҲ йҷӨдҫқиө–йҮҚиЈ…,еҸҜд»Ҙи®©е…¶зӣҙжҺҘ жү§иЎҢ patch-packageеҚіеҸҜ

жҖ»з»“дёӢ:

з”ҹжҲҗиЎҘдёҒ: `npx patch-package еҢ…еҗҚ`
жү§иЎҢиЎҘдёҒ: `npx patch-package`

жіЁж„Ҹ

  1. npx patch-package еҢ…еҗҚз”ҹжҲҗиЎҘдёҒеҺҹзҗҶжҳҜе…ҲеҺ»package.jsonе№¶дё”еҝ…йЎ»иҰҒжңүpackage-lock.json(жҲ–иҖ…yarn.lockд№ҹеҸҜд»Ҙ)дёӯжүҫеҜ№еә”зҡ„еҢ…еҗҚ,з„¶еҗҺеҶҚеҺ» node_modulesж–Үд»¶дёӯжүҫ,жүҖд»ҘеҰӮжһңpackage.jsonжҲ–иҖ…package-lock.jsonж–Үд»¶жІЎжңү,е°ұдјҡжҠҘй”ҷжүҫдёҚеҲ°
  2. з”ҹжҲҗзҡ„иЎҘдёҒж–Үд»¶е®һйҷ…дёҠжҳҜдёҖдёӘdiffж–Үд»¶пјҢдҪҝз”Ёnpx patch-packageжү§иЎҢиЎҘдёҒж—¶,
    е…¶дјҡиҮӘеҠЁж №жҚ®patchж–Үд»¶зҡ„еҗҚеӯ—(еҢ…еҗҚ+зүҲжң¬еҸ·)зӣҙжҺҘеҺ»node_modulesж–Үд»¶дёӯжүҫ,еҰӮжһңжүҫдёҚеҲ°д№ҹдјҡжҠҘй”ҷ, зүҲжң¬еҸ·дёҚеҜ№д№ҹдјҡжҠҘй”ҷ.
    дёҚдјҡдҫқиө–package.jsonе’Ңpackage-lock.jsonж–Үд»¶.
    жүҖд»ҘеҒҮеҰӮз”ҹжҲҗpatchзҡ„зүҲжң¬жҳҜ 2.5.10зүҲжң¬,еҒҮеҰӮз¬¬дёүж–№жӣҙж–°дёә2.5.11ж—¶, жү§иЎҢиЎҘдёҒдјҡжҸҗзӨәжүҫдёҚеҲ°.й—®жҳҜеҗҰжҳҜ2.5.10,жӯӨж—¶еҶҚжүӢеҠЁжү§иЎҢnpx patch-package element-ui, жӯӨж—¶patchж–Үд»¶еҗҚдјҡиҮӘеҠЁеҸҳжӣҙдёәдёә2.5.11зүҲжң¬, дҪҶжҳҜжіЁж„ҸеҫҲжңүеҸҜиғҪеҚідҪҝиЎҘдёҒжҲҗеҠҹеә”з”ЁпјҢд№ҹдјҡеҮәзҺ°з ҙжҚҹ,жүҖд»ҘиҮӘе·ұиҰҒж ёеҜ№дёӢ
