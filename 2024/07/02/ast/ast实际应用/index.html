我们知道:箭头函数没有this,他的this就是会一层层往上找最近作用域的this
所以我们要实现

  1. 在最近作用域创建var _this = this;
  2. 修改箭头函数的所有用到this的地方为_this

实现

找到箭头函数的上级作用域

  • 找作用域processFunctionScope: 上级是函数且不是箭头函数或者是最外层根节点
  • 找到所有箭头函数用到this地方放到数组中: getThisPaths方法
  • 遍历上面this数组,替换this为_this: replaceWith方法
  • 在作用域上创建var _this=this: scope.push方法

实战5-babel-plugin-import按需加载

import {flatten,concat} from 'lodash'转换为
import flatten from "lodash/flatten";import concat from "lodash/concat";

  • 因为lodash直接加载,会把所有包都加载进来,我们可以按上面方式实现按需加载
  • 注意⚠️: 该方法的前提是组件必须拆出来一个个模块. lodash、ant、element、vant都是这样子的.
  • 上面就是 babel-plugin-import的原理, 一般组件库实现按需都是这样拆出来一个个模块的.
