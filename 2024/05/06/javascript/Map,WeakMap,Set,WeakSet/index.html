Map,WeakMap,Set,WeakSet

发表于2024-05-06|更新于2024-05-06|javascript傻傻分不清楚

Map

由于js对象的键只能是字符串,如果硬要对象作键, 键会变成'[object Object]', 这给使用带来了很大的限制,所以es6新增Map, 任意类型都可以当键

WeakMap

  • 主要特点是,键只能是对象如对象、数组、函数(null除外)和symbol,不能是基本类型,打印结果和map格式一样
  • 键是弱引用,不计入垃圾回收机制, 但值仍是强引用,计入垃圾回收
  • 只有增删改查,没有遍历、size, 因为是弱引用,不知道浏览器什么时候回收,所以也就不可能存在遍历,size

WeakSet

  • 和set一样,也会自动去重、只能添加对象如对象、数组、函数(null除外)和symbol,不能是基本类型,打印结果和set格式一样
  • 是弱引用,不计入垃圾回收机制
  • 只有增删改查,没有遍历、size, 因为是弱引用,不知道浏览器什么时候回收,所以也就不可能存在遍历,size

总结

  • map是添加([key,value]) 对标对象, set是添加([value,value])对标数组,原型上方法是增删改查、size、遍历
  • weakMap和weakSet和上面格式一样, 但是是弱引用, 注意weakMap键名是弱引用,值是强引用, 原型上方法没有遍历、size
  • set和weakSet会自带去重
  • 这四个类型对于NaN,都认为是一个键
  • map的新增是set,set的新增是add
  • 这四个类型不是js新增的基本数据类型,它们都是 ES6 引入的新的数据结构，被称为“集合类型”（Collection types）或“内置对象”，属于复杂数据类型或引用类型。
